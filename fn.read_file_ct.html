pub fn read_file_ct(path: &Path) -> Result<Vec<u8>>
Expand description

Read an entire file while using the same code path on success or failure.

This aims to keep timing similar for error and success cases but it is not a strong constant-time guarantee. True constant-time I/O would require operating system support and is outside the scope of this crate.

Returns Error if the file cannot be read.
